- 获取热门榜单

```
传递参数limit=返回数据条数
// 日榜
http://www.app-echo.com/api/rank/sound-hot?periods=daily&limit=12

// 周榜
http://www.app-echo.com/api/rank/sound-hot?periods=weekly&limit=12

// 月榜
http://www.app-echo.com/api/rank/sound-hot?periods=monthly&limit=12

```
